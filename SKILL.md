# Usage

Claude API rate limit 사용량과 OpenCode 세션 통계를 확인하는 커맨드.

## 실행 방법

아래 셸 스크립트를 실행하고, 출력 결과를 코드블록으로 표시하세요:

```bash
bash ~/.config/opencode/skill/usage/scripts/usage.sh
```

출력 결과를 그대로 코드블록 안에 넣어 사용자에게 보여주세요.
스크립트 에러 시에도 에러 메시지를 그대로 전달하세요.
