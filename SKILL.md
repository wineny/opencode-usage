# Usage

Claude API rate limit 사용량과 계정 정보를 확인하는 커맨드.

## 실행 방법

`bash ~/.config/opencode/skills/usage-claudecode/scripts/usage.sh` 를 실행하세요.

## 출력 규칙

- 스크립트 실행 결과(stdout)만 코드블록 없이 그대로 사용자에게 전달하세요.
- 실행 과정, 명령어, 부가 설명을 추가하지 마세요.
- 에러 발생 시에도 에러 메시지만 그대로 전달하세요.
